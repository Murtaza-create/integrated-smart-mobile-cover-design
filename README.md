Integrated Smart Mobile Cover with Built-in Power Bank, Cooling Fan & LED Torch
Project Overview

This project presents the conceptual design and development of a multi-functional smartphone cover that integrates three practical features into a single compact accessory:

Portable Power Bank
Cooling Fan
LED Torch

The objective was to improve smartphone usability by combining charging, thermal management, and emergency lighting into one ergonomic design while maintaining portability and ease of use. The presentation outlines the integrated concept, component selection, workbenches involved, design considerations, limitations, and engineering challenges.

Objectives
Design a compact multi-functional smartphone cover.
Integrate a rechargeable power bank into the cover.
Improve smartphone thermal performance using an integrated cooling fan.
Provide an emergency LED torch for additional functionality.
Apply engineering design principles to consumer electronics.
Key Features
🔋 Built-in Power Bank for portable charging
🌬️ Integrated Cooling Fan to reduce device overheating
🔦 High-efficiency LED Torch
📱 Compact and ergonomic enclosure
🛡️ Protective smartphone casing
⚡ Power management circuitry
🌡️ Thermal management considerations

These core features are described throughout the project presentation.
